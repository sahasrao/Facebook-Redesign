# Facebook Redesign

A responsive Facebook UI redesign built with React, TypeScript, and Tailwind CSS.

## Features

- **Home Feed** — Post composer, news feed with images, like/comment/share actions
- **Profile** — Cover photo, profile info, about section, and posts
- **Friends** — Friend cards with mutual friends and messaging shortcuts
- **Community** — Featured groups, group list, and create group CTA
- **Responsive Layout** — Three-column desktop layout; mobile bottom navigation; collapsible sidebars on tablet

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- React Router

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home feed |
| `/friends` | Friends list |
| `/community` | Groups & community |
| `/profile` | User profile |
